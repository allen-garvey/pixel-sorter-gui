# Pixel Sorter

GUI application made with Qt to pixel sort image files. Based on [Qt ImageView example](http://doc.qt.io/qt-5/qtwidgets-widgets-imageviewer-example.html)

## Dependencies

* Qt 5.8.0
* g++ 5.4.0
* OpenMP

## Getting Started

* Open project in Qt Creator
* Run project
